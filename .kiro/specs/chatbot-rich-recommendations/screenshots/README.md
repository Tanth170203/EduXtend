## Screenshot Guidelines

### Capturing Screenshots

**Desktop:**
- Use 1920x1080 resolution
- Capture full browser window or crop to chatbot area
- Ensure UI is clean (no personal data visible)
- Use Chrome or Firefox for consistency

**Mobile:**
- Use iPhone 12/13 or Samsung Galaxy S21 size
- Capture full screen or crop to relevant area
- Use device simulator or actual device
- Ensure touch targets are visible

### Image Specifications

- **Format**: PNG (for transparency and quality)
- **Resolution**: High DPI (2x for retina displays)
- **File Size**: Optimize to < 500KB per image
- **Naming**: Use kebab-case as specified above

### Content Guidelines

**Do:**
- Use realistic but anonymized data
- Show complete, functional UI
- Capture clean, professional screenshots
- Include relevant context (chat history)
- Show actual Vietnamese text

**Don't:**
- Include personal student information
- Show error states (unless documenting errors)
- Use Lorem Ipsum or placeholder text
- Capture during loading states
- Include browser UI unless relevant

## Taking Screenshots

### Step-by-Step Process

1. **Set up environment:**
   - Clear browser cache
   - Use incognito/private mode
   - Set appropriate screen resolution
   - Ensure chatbot is working properly

2. **Prepare test data:**
   - Use sample data provided above
   - Ensure realistic Vietnamese text
   - Verify all card elements display correctly

3. **Capture screenshots:**
   - Use built-in screenshot tools or Snipping Tool
   - Capture at 2x resolution for retina displays
   - Save as PNG format

4. **Edit if needed:**
   - Crop to relevant area
   - Blur any sensitive information
   - Optimize file size
   - Verify image quality

5. **Save with correct naming:**
   - Use exact filenames specified above
   - Save in this directory
   - Verify file size < 500KB

6. **Update documentation:**
   - Replace placeholder text in USER_GUIDE.md
   - Add actual image references
   - Test that images display correctly
